# 内网隧道代理工具
### 测试环境：
  Windows 11 x64
  
  Linux kali (debian) x64

使用方法：
-
    Server:
      windows : Socket-Server.exe [ port ]
      Linux   : ./Socket-Server [ port ]
      
    Client:
      Windows : Socket-Client.exe [ serverip:serverport ] [ proxyPort ]

    本地浏览器 挂代理：
      地址：127.0.0.1
      端口：proxyPort
      
    Proxifier 同上


说明：
-
    各位大佬，工具目前 只能把 Web 流量带出来 后续升级版本 会带出其他流量，感谢关注
